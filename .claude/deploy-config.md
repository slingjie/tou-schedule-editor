# Cloudflare Pages 部署配置
# 使用方法：
# 1. 获取 API Token: https://dash.cloudflare.com/profile/api-tokens
#    需要权限: Cloudflare Pages > Edit
# 2. 设置环境变量: export CLOUDFLARE_API_TOKEN=your_token
# 3. 运行部署: npx wrangler pages deploy dist --project-name=tou-schedule-editor

# 或者直接使用以下命令部署：
# npx wrangler pages deploy dist --project-name=tou-schedule-editor --branch=main
